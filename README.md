# flutter music

一个自己拿来练手的flutter项目，仿照网易云音乐客户端编写，

## Getting Started

- 下载源代码

```$xslt
git clone https://github.com/dczxw/flutter_music.git
```

- 安装依赖
```$xslt
flutter packages get
```
